# Codidot Dynamic Rates Plugin
A professional Elementor Addon for an International Rates Calculator with Bulk CSV Upload functionality.

## Author
Ajay Yadav | Codidot IT Solutions

## Features
- **Bulk CSV Upload:** Upload thousands of rates instantly.
- **Manual Management:** Add or delete rates manually from the backend.
- **Elementor Widget:** Simple drag-and-drop widget for Elementor pages.
- **AJAX Calculation:** Real-time rate calculation without page reloads.

## Installation
1. Upload the `codidot-dynamic-rates` folder to your `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Add rates via the new **Codidot Rates** menu.
4. Edit any page with Elementor and drag the **Codidot Rates Calculator** widget onto your page.

# Contact Form 7 Lead Management System

This feature allows all submissions from **Contact Form 7** to be automatically saved in the WordPress backend for easy lead management and future access.

## Features

### 1. Save Form Submissions
All form entries submitted through Contact Form 7 are automatically stored in the database.

- No lead data is lost
- Entries remain محفوظ even if email notifications fail
- All submissions are stored securely

### 2. Admin Dashboard Access
A dedicated admin panel is available in the WordPress backend where all leads can be viewed.

Admin can see details such as:

- Name
- Email Address
- Phone Number
- Message
- Date & Time of Submission

### 3. Lead Management
The admin can easily manage all submitted leads from a single dashboard.

- View all entries
- Search and filter leads
- Organize submissions efficiently

### 4. Export Leads
The admin can download all leads in the following formats:

- **CSV Format** (`.csv`)
- **Excel Format** (`.xlsx`)

This makes it easy to use the data for reporting, follow-up, or sharing with the team.

### 5. User-Friendly Interface
A clean and simple backend interface ensures easy navigation and efficient lead handling.
